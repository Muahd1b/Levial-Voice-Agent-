# Utility Scripts

This directory contains utility scripts for maintaining and managing the Levial Voice Agent.

## Scripts

### download_models.py

Downloads required model files for Piper TTS.

**Usage:**

```bash
python scripts/download_models.py
```

### verification_script.py

Verifies the installation and configuration of the voice agent, checking:

- Python dependencies
- Model file locations
- Configuration validity
- Directory structure

**Usage:**

```bash
python scripts/verification_script.py
```
